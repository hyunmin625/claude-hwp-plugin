# HWP Agent 개발 계획

## 프로젝트 목표

**Claude Code Plugin(Skill)** 형태로 배포되어, Claude Code를 통해 한글워드 문서 파일(.hwp, .hwpx)을 완전히 제어하는 AI 에이전트 도구를 구현한다.

- 최종 산출물: Claude Code에서 설치·사용 가능한 `.plugin` (skill 포함)
- 핵심 가치: 자연어 명령만으로 HWP/HWPX 문서의 생성·읽기·수정·저장을 자동화
- 참조 구현: GitHub **kordoc** 프로젝트의 파일 처리 방식 및 API 설계를 기반으로 개발

---

## Plugin 구조 설계

### 구성 요소
- **MCP Server**: HWP/HWPX 파일 조작을 담당하는 백엔드 서버 (Python 기반)
  - kordoc 프로젝트의 파싱/쓰기 로직을 래핑하여 MCP 도구로 노출
- **Skill**: Claude Code 내에서 사용자 자연어 명령을 MCP 도구 호출로 변환하는 프롬프트·워크플로우
- **Plugin 패키지**: MCP Server + Skill을 묶어 `.plugin` 파일로 배포

### 제어 범위 (완전 제어 목표)
| 기능 | 설명 |
|------|------|
| 읽기 | HWP/HWPX 문서의 텍스트, 표, 이미지, 메타데이터 추출 |
| 쓰기 | 새 문서 생성, 텍스트·표·이미지 삽입 |
| 수정 | 기존 문서의 특정 위치 내용 변경, 스타일 적용 |
| 양식 채우기 | 템플릿 문서에 데이터를 자동 매핑하여 완성 문서 생성 |
| 저장 | .hwp / .hwpx 포맷으로 저장 및 내보내기 |

## 기능 목록

### 1. HWP/HWPX 양식 자동 채우기 (Auto Form Fill)
- **개요**: HWP 또는 HWPX 한글워드 문서에 미리 작성된 양식(템플릿)이 있을 경우, 사용자에게 필요한 데이터를 입력받아 AI 에이전트가 자동으로 해당 양식의 내용을 채워주는 기능
- **주요 흐름**:
  1. 사용자가 HWP/HWPX 양식 파일 업로드
  2. AI 에이전트가 양식 구조 분석 (입력 필드, 표, 반복 항목 등 파악)
  3. 사용자에게 필요한 데이터 요청 (직접 입력 또는 데이터 파일 업로드)
  4. AI 에이전트가 데이터를 양식에 맞게 자동 매핑 및 채우기
  5. 완성된 HWP/HWPX 파일 출력
- **고려 사항**:
  - 표, 텍스트 필드, 반복 구조 등 다양한 양식 유형 지원
  - 대량 데이터 처리 시 일괄 생성(batch) 지원 여부
  - 한글 인코딩 및 폰트 유지

## 개발 단계 (Roadmap)

### Phase 1 — 기반 구축
- kordoc 프로젝트 분석 및 핵심 모듈 파악
- HWP/HWPX 읽기/쓰기 기본 기능 구현 (MCP 도구로 래핑)
- Claude Code에서 MCP 연결 및 기본 동작 검증

### Phase 2 — 핵심 기능 구현
- 전체 제어 기능(읽기·쓰기·수정·저장) 구현
- 양식 자동 채우기(Auto Form Fill) 기능 구현
- Skill 프롬프트 작성 (자연어 → MCP 도구 호출 변환)

### Phase 3 — Plugin 패키징 및 배포
- MCP Server + Skill을 `.plugin` 파일로 패키징
- 설치 가이드 및 사용 예시 문서화
- 테스트 및 엣지케이스 대응 (대용량 문서, 복잡한 표 구조 등)

## 구현 참조 자료

### GitHub - kordoc 프로젝트
- **참조 링크**: https://github.com/search?q=kordoc&type=repositories (검색 후 해당 프로젝트 확인)
- **활용 목적**: MCP Server 구현 시 kordoc 프로젝트의 구조, API 설계, 파일 처리 방식을 기반으로 개발
- **참조 대상**:
  - HWP/HWPX 문서 파싱 및 쓰기 로직
  - 문서 구조(단락, 표, 필드 등) 접근 방식
  - 한글 문서 포맷 처리 관련 유틸리티 및 헬퍼 함수
- **적용 방법**:
  1. kordoc 프로젝트 코드 분석 후 핵심 모듈 파악
  2. MCP Server 개발 시 kordoc의 문서 조작 API를 기반으로 래핑(wrapping)
  3. AI 에이전트가 자연어 명령을 MCP 도구 호출로 변환하는 Skill 설계
